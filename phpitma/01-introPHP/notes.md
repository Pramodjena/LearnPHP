# What is web development ?

- Web development is the process of creating applications that runs on the internet using web technologies.

It includes : 

-  Frontend(client side) 
-  Backend(server side)
-  Database 

# Components of web development :

1. Frontend(HTML,CSS,JAVASCRIPT)
    - Responsible for : 
        - Design 
        - Layout
        - User interface
2. Backend(PHP,JAVA,PYTHON,NODE.JS)
    - Runs on server 
        - Processing data 
        - Handling authentication 
        - Connecting to databse 
        - Business logic
3. Database(MYSQL,MONGODB,POSTGRESQL)
    
    - Stores data permanently 

# What is server side technology ?
 
- Server side technology refers to programming that executes on the web server before sending response to the client.

- THe browser never sees server-side code - it only shows the output.

## Why do we need server-side ?
 - Runs on server 
        - Processing data 
        - Handling authentication 
        - Connecting to database 
        - Business logic  
        - Perform calculations securely 

## Client vs Server 

  Client :
    A client is a device or browser that sends a request to a server.
    
    Examples :
      - Chrome 
      - Brave 
      - Firefox 


Server :
   A server is a system that : 
      - Receives request 
      - Processes it
      - Sends response 

## How communication happens :
 - User types domain name 
 - ISP 
 - DNS 
 - Web server(Server handles HTTP request)
 - Server processes it 
 - Server sends response based on the request
 - Browser displays the output

URL - Uniform resource locator (Address of a webpage)
HTTP - Hyper text transfer protocol(Used for communication b/w client and server)

 ## What is PHP ?

- PHP : Hypertext Preprocessor 
  - Server side scripting language 
  - Used to develop dynamic website/apps
  - Embedded inside HTML 
  - Free and Open source
  - Platform Independent 
  - Support major database
  - Easy to learn language
  - Large community support 

## Use cases of PHP 
- Login systems  
- Form processing 
- E-comm website 
- Content management system(wordpress)
- Online result ports 


- Web server(Apache, Nginx, IIS)
