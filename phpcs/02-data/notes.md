## What is  PHP ?

- PHP: stands for php hypertext preprocessor 

- Server side scripting language 
- Used to develop dynamic web apps 
- Embedded  inside HTML 
- Open source
- Platform independent 
- Easy to learn 
- Large community support 

## Use case :

- Login system 
- Form processing 
- E-comm website 
- Content management system(wordpress) 
- Online result portals

## Variable 

- Must start with $  
- Cannot start with number 
- Case-sensitive(age != Age)
- No space allowed 
- No special char(_)

## Data types :
 - Integer(-ve, +ve , 0 no float numbers) 
 - Float(-0.5,1.5 etc..)
 - String("any name",'any name')
 - Boolean(true,false)
 - Array($color = array("red","blue"))
 - NULL
